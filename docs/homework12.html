Question 1: Choropleth map of GDP in Europe

### load data here:
gap = px.data.gapminder()

gap.head()

gap_eu_1997 = gap.query("continent=='Europe'").query("year==1997")

fig_eu_1997 = px.choropleth(
    gap_eu_1997,
    locations = "iso_alpha",
    color = "gdpPercap",
    hover_name = "country",
    hover_data = {
        #"lifeExp":":1f",
        "pop":":,",
        "gdpPercap":":,.0f",
        "iso_alpha":False
    },
    color_continuous_scale="Viridis",
    title = "GDP per capita by country",
    subtitle = "Europe, 1997",
    scope = "europe"
)

fig_eu_1997.show()
# I added 'scope' to show only Europe, and I added 'subtitle' to clarify that this data was showing Europe in 1997

Question 2: Bubble map of population

bubblemap = px.scatter_geo(
    gap.query("year==1997"),
    locations="iso_alpha",
    size="pop",
    color="continent",
    hover_name="country",
    hover_data = {
        "lifeExp":":1f",
        "pop":":,",
        "iso_alpha":False
    },
    title = "Population size by country",
    subtitle = "Global, 1997",
    projection = "wiechel"
)

bubblemap.show()

# I added a subtitle to clarify year, and changed the projection to 'wiechel', which turns the map into a kaleidoscope.
